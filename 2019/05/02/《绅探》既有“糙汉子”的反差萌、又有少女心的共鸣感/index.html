《绅探》既有“糙汉子”的反差萌、又有少女心的共鸣感

发表于 2019-05-02 | 阅读次数：
《绅探》

点击上方蓝字▲ 关注订阅喵星视界 欢迎转发到朋友圈

如今网剧的制作水平也提高了不少，甚至有些网剧比上星剧还要精良，还要好看。今天要说的《绅探》就是这样一部引发广泛关注网剧，这部剧主演的演技都不错，剧情紧凑不拖沓，服道化也是用心的，如果一定要说的话就是案情不够细致，不太经得起推敲。

该剧讲述了民国绅士侦探罗非（白宇饰）与热血女警秦小曼（尤靖茹饰）联手与幕后黑暗组织斗智斗勇的故事。

白宇前两年的发展并不理想，虽然有颜值有演技，但是却一直演一些不重要的边边角角，然而让他大红大火的却依然是《镇魂》这部剧。

白宇饰演的神探罗非，在形象上，颠覆了以往的“糙汉feel”，走起了走起了精致风，无论是西装革履的民国复古搭配，还是随身携带的神探必备单品，如文明仗、怀表、雨伞等，举手投足都是“绅士范”。

在剧中，尤靖茹饰演的秦小曼，刚从警校毕业的秦小曼怀着成为一名出色警探的理想，来到英租界，住进了沙利文公寓，与屡破奇案、名声大噪的警局探案顾问神探罗非（白宇饰），成了邻居和同事。

作为巡捕房中唯一的女警探，不仅以全优的成绩从警校毕业，而且在实际破案过程中，经常能够发现至关重要的线索，从而推动整个情节的发展。

现在市面上的悬疑剧，无论是男女主还是双男主，哪个不是这样的搭配？

经过一次次的惊险破案后，铁三角逐渐发现隐藏在上海滩之下的庞大黑暗组织，与真凶也上演一场猫捉老鼠的精彩对决。

罗非是探案时当之无愧的“高智商担当”，在破案时逻辑严密，却偶尔“戏精”附体不乏笑点，与搭档相处时“口嫌体正”，时常情商下线。女警秦小曼初出茅庐、工作经验不足，但武力值爆表。

身为“武力担当”，小曼常常在力量和格斗中，对男性对手进行无情碾压，动作干脆、身型利落，令观众不禁大呼“小曼好帅!小曼好Man!”

尤其是当秦小曼挺身而出，“美救英雄”，为侦探罗非的安全保驾护航时，更是让观众纷纷感慨：“秦小曼这是拿了男主剧本吧？!”

虽然经常被罗非嘲讽无脑，但秦小曼并不是那种只会尖叫和提问的傻白甜女主，她是剧里的武力担当，需要跟职业杀手单挑的那种，女主角的设置，让这部剧好看了很多。

两个人性格一定互补，一定是“欢喜冤家”，罗非雅痞孤傲、口嫌体正，探案时专业专注，秦小曼热血外向，一边跟着罗非慢慢成长，一边慢慢打开罗非封闭的内心。

法医本杰明虽性格高冷，但爱岗敬业，乐于以实验室为家，各项医疗鉴定技能满点。由犯罪心理专家、警察队长及法医等类型化职业，自由组合并担任功能性角色。虽然主角视角有所不同，但多数情况下设定一位职场小白，以增强观众的代入感。

每个角色都有属于自己的人生，她们之中有的追求自己的进步理想，有的期望一份不容于世爱情，有的享受快意恩仇的自在生活，有的仅仅是想要有尊严地活着。

《绅探》向我们展现出了一个个鲜活、饱满的民国女性形象，展现出了不同的风情和故事性。

美而不同，她们都是自己人生的主角。

女主虽然心怀单纯理念、有过度理想化的高光痕迹，但剧作巧妙避免了她向“圣母女表”方向发展的尴尬。

云，月，梦，雪，四姐妹，可真应了那句：一片寒微骨，翻成面面心。自从遭点染，抛掷到如今。

愤怒的海棠花里，出现的所有女性角色，包括小孩子。选角真是用心呀，就连齐雪儿役演员和齐云的女儿都选得很恰当，齐妈在破落的满庭芳披头散发地说有客人了，颇有种何处话凄凉的时代落寞之感。

在追寻真相的过程中，女警探秦小曼一路成长不断，升级打怪，凭借着多次发现关键线索，不仅助力剧情发展，更是让一贯骄傲的天才罗非也渐渐刮目相看。

作为一部讲述发生在三十年代上海滩背景下的破案题材的电视剧，《绅探》在服化道上都力求真实还原上世纪的国际摩登大都市。

全剧在采用了大量海派美学体系，场景与服装上的配合，道具与人物性格特点的结合，无一不突显着制作团队的用心，受到了广大网友的夸赞。

每一件离奇的案件，往往伴随着高超的探案推理过程。在《绅探》中，可能看似是一桩简单普通的案件，结果却一波三折、出人意料。观众们可以跟随主演们缜密的推理过程，了解案件的真相与事实。

整个画面的色调非常沉稳，是慢慢的民国感，人物在画面中也非常的有格调。总之，终于让大家告别了阿宝色，大家还是很开心的。然后就是配乐，气氛一松一弛，配乐也随之变换。很好地烘托了氛围。

有什么宝贵的**意见，或者想要小编写什么，你说，小编会慢慢看的，虽然不一定听，哈哈哈哈。**
