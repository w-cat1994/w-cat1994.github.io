《破冰行动》能猜到谁是最终大坏蛋吗？人性啊，真的超然的

发表于 2019-05-18 | 阅读次数：
《破冰行动》大尺度、警匪、毒贩

点击上方蓝字▲ 关注订阅喵星视界 欢迎转发到朋友圈

《破冰行动》正在热播中，这部缉毒剧还是蛮不错的，无论是剧情还是演员都非常有吸引力，而且这部剧还是根据真实案件改编的，很有影响力，对观众而言，也是近期难得的一部好作品。

该剧是有现实原型的，根据广东雷霆扫毒“12.29专项行动”改编，是一个真的不能再真的扫毒大案，当时警方歼灭的“制毒第一村博社村”，就是剧中的塔寨村，开头村民围堵警察的画面，正是真实发生过的事情。

所以开播以来不仅获得了超高的话题热度，更有大批观众被直击人心的剧情实力圈粉，纷纷表示对这部真实燃向的国情剧毫无抵抗能力，重新燃起了“追剧”的快感和期待。

这部剧的好看之处就是情节不拖泥带水，节奏紧凑，该剧又是根据真实故事改编，写实感很强，这样一来观众很快就能深入其中。

要说谁是坏人，就不免要观察的细致一些了，很多隐藏在幕后的人可能会让大家大吃一惊。蔡队其实什么都看得通透！就因为看得通透，所以只能履行上边的意思行事！能在善与恶之间保持自己与家人的平安，同时也没被孵化，很难得了！他才是最有能力的人。

博社村的人，基本都姓蔡，村原党支部书记蔡东家（已于2019年1月17日被执行死刑），即是《破冰行动》中塔寨村老支书林耀东的原型。博社村的制毒贩毒，也基本只让蔡姓同族参与，由此，便形成以蔡东家为首的宗族制贩毒网络，蔡东家也因此被称为“冰毒教父”。

上至市长、厅长，下至贩毒集团喽啰，接下来，便是“闭眼”、“睁眼”、“验人”、“投票”等一系列迷雾重重的狼人杀过程，谁是好人，谁是坏人，谁站在黑白之间的灰色地带，观众都可以依据剧情推演、浮上明面的人物关系，甚至人物动作语言细节，进行自己的推论，从而享受推理乐趣。

网播对叙事节奏要求更严苛，而台播则更注重政策性导向，比如第二集开场，网播版中，主角李飞直接持枪进入养鸡场毒品交易地，并迅速与毒贩交火，而台播版中，则是省厅禁毒局副局长李维民主持工作的场景。

用拍纪实片的手法拍电视剧，或者用拍电视剧的手法拍纪实片，不管哪样，不把观众当傻瓜的好剧组才有值得称许的产出。

吴刚、王劲松、任达华等老戏骨与“鲜肉演员”黄景瑜飙戏；拍摄得到公安部大力支持；这些，也都成为此剧重要看点。

警匪之间、警界内部，都呈现出极为动人的戏剧张力，而相比李飞、宋杨、蔡军等基层警员制造出的表面冲突，高层权力者一旦博弈，呈现给观众的，便是阅历丰富、以轻击重的深层暗涌之力，这种力量往往通过微表情、无起伏对话等绵密的过招方式得以被展现。

这部剧中隐藏的其他势力，可以说不止是塔寨村这一关势力，大大小小的实力错综复杂，例如南井村、香港、澳门等地。

一个仅有两千多户的村庄却有40座祠堂（祠堂是宗族制的典型文化载体），两千多栋房屋紧紧挨着，各家各户也没有门牌号，小道如麻，大车无法进出，密合宛如堡垒，剧中林胜文甚至敢在自家厨房公然制毒。

缉毒警察只要一进来，如果不了解村中复杂的道路情况，能轻易被村民围死，而李飞和宋杨仅仅带了几个外地警察就敢闯村，所以蔡永强才会朝李飞大发雷霆。

太真实了！剧中塔寨村非常还原博社村实景，李飞抓捕林胜文时就是躲进了林氏宗族祠堂。

《破冰行动》中，无论身份还是手腕，李维民都是一等一的高手，他的提问思路清晰、主题集中、火力充足。

李维民和李飞的父子线，是全剧的一大看点。从小被李维民收养的李飞，受到父亲的影响，最终选择了和李维民一样的缉毒道路。

外表稳如泰山、心里慌到不行的李维民和不服就是干、热血闯天下的李飞在办案方面有着极大的不同。

而蔡永强与其对阵，丝毫不输，他的回答，既不正面也不反面，而且自始至终保有警察的基本职业道德，同时在不确定李维民立场的情况下，天衣无缝地隐藏了自己的立场。

这伟大背后的不舍和愧疚或许能更深入人心，画面的刺激和情节的紧张，或许能给观众带来一定的爽感，却很难给观众以心灵的激荡，而在这样的人性讨论面前，悬念感和惊悚就已经被弱化了。

王劲松饰演的林耀东是本剧中最大的反派，从第一幕登场开始，就阴沉着一张脸。尽管在《破冰行动》前几集里他的戏份不多，但我相信这个反派形象一定会在后面大放异彩。

马云波和蔡永强之间一定有一个人是反派，但是细节――蔡永强故意放李飞从天花板上离开自己毒贩说“有人不让动李飞”，李飞和马云波之间又有过命的交情这一渊源，马云波反派的可能性更大一些。

蔡永强不信任李飞，他也不信任陈光荣，他才是一个狠人！“不是每个人都知道真相，但是每个人都能成为真相”，蔡永强心思缜密，滴水不漏。

赵嘉良可以说是最大的卧底了，他和李维民是几十年的兄弟，而且还是李飞的亲生父亲，他成为香港的毒枭，其实都是为了给自己的妻子复仇，而且也是未来自己的儿子，这么多年来一直漂泊在外。

确实蛮心酸的，不过结局的最后，赵嘉良还是有个圆满的结局，他帮助警察破案了，也能够回国和李飞以及李维民团聚，也算是很好的结尾了。

很久没有看到这么畅快淋漓的硬汉剧情，没有狗血的爱情套路，真相抽丝剥茧，迷雾重重，期待后续情节，抓出狼王的那一刻，一定是最激动人心的时刻。

演员和观众却以一种形式来向他们表示出一种“无声的致敬”即敢说,敢做,敢拍,为正义加持,让信仰永不落幕。

有什么宝贵的**意见，或者想要小编写什么，你说，小编会慢慢看的，虽然不一定听，哈哈哈哈。**
